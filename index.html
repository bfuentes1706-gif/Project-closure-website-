Title: Adjusting Our Project When New Ideas Came Up

Situation

My team was building a small web app for our semester project. After we showed the first version to our group and got feedback, a few teammates suggested adding extra features like filters and a more detailed page layout. These weren‚Äôt part of our original plan and would take more time.

Task

I needed to help the team figure out how these new ideas would affect our timeline and decide which ones we could realistically include.

Result

We ended up keeping just the filter feature for this version and setting the rest aside for ‚Äúfuture improvements.‚Äù We finished the project on time and still improved the user experience. I learned that not every idea needs to fit into the current version‚Äîsometimes you have to prioritize.
